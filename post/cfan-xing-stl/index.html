C++泛型/STL

2020-11-28

函数模板
格式：
template //或写成与class无区别
//需要不同类型参数时<typename T1,typename T2>u77
返回值 函数名 （形参表）
{
函数体；
}
template
T Max(T Inpute1, T Inpuet2)
{
return (Inpute1 > Inpute2) ? Inpute1 : Inpute2;
}
使用：
1.自动类型推导
Max(a,b);
2.显示指定类型
Max(a,b);
类模板
template
class 类名
{
类定义；
}
在类外定义成员函数时：
template
返回值类型 类名::函数名(T x,T y)
{
函数体；
}
类模板声明对象：
类名<泛型数据类型> 对象名；
